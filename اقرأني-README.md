# كيف تفعّلي لوحة التحكم (بدون أي برمجة)

هاي الطريقة الوحيدة اللي بتخليكي تضيفي أخبار وفرص بنفسك من متصفح عادي، بدون ما تلمسي كود، وبدون ما تدفعي مصاري.

## الخطوات (مرة وحدة بس، أول ما تسوّيها بتشتغل للأبد)

**1. سوّي حساب GitHub (لو ما عندكم)**
اذهبي إلى github.com وسجّلي حساب مجاني.
- ارفعي مجلد هاد كامل (بكل الملفات: index.html، مجلد admin، مجلد content) كـ "repository" جديد اسمه مثلًا `forena-site`.
- لو ما بتعرفي ترفعي ملفات على GitHub، في زر "Add file → Upload files" برواجهة الموقع، بس اسحبي كل الملفات وحطيها هناك.

**2. اربطي الموقع بـ Netlify (استضافة مجانية)**
- اذهبي إلى netlify.com وسجّلي حساب مجاني (فيه خيار "Sign up with GitHub" وهو الأسهل).
- اضغطي "Add new site" → "Import an existing project" → اختاري GitHub → اختاري الـ repository يلي رفعتيه.
- اتركي كل الإعدادات كما هي واضغطي "Deploy". بعد دقيقة رح يعطيكي رابط للموقع شغّال مباشرة (مثلاً forena-site.netlify.app)، وفيما بعد فيكي تربطي دومين خاص فيكم عادي من نفس اللوحة.

**3. فعّلي "Identity" و "Git Gateway"**
هاي الخطوة يلي بتفعّل تسجيل الدخول للوحة التحكم.
- من لوحة Netlify لموقعك: Site configuration → Identity → فعّليها (Enable Identity).
- تحت "Registration preferences" اختاري "Invite only" (أأمن، بس أنتم يلي تدعو نفسكم).
- روحي لـ Identity → Services → Git Gateway → فعّليها (Enable Git Gateway).
- ارجعي لـ Identity → ادعي نفسك (Invite users) بإيميلك، رح يوصلك إيميل فيه رابط.

**4. افتحي لوحة التحكم**
- من الإيميل يلي وصلك اضغطي الرابط وسوّي كلمة سر.
- بعدين روحي لـ `الرابط-تبع-موقعك/admin` (مثلًا forena-site.netlify.app/admin) وسجّلي دخول.
- رح تشوفي قسمين: "الأخبار" و"الفرص" — اضغطي عليهم وضيفي/عدّلي/احذفي بكل سهولة من نموذج بسيط.
- كل ما تحفظي (Publish)، التغيير بيظهر على الموقع الحقيقي خلال دقيقة أو دقيقتين تلقائيًا.

## ملاحظات مهمة
- ما في داعي تفتحي أي كود؛ اللوحة شكلها نموذج عادي (عنوان، تاريخ، تفاصيل، رابط).
- كل من تدعيه من "Invite users" رح يقدر يدخل يعدّل، فادعي بس الأشخاص الموثوقين.
- إذا حبيتي لاحقًا تضيفي أقسام تانية غير الأخبار والفرص (متعدّلها من اللوحة)، خبريني وأنا بضيفها.
